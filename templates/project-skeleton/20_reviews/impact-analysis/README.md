# Impact Analysis (20_reviews/impact-analysis)

이 폴더는 **Impact Analyzer Agent**의 분석 결과를 저장합니다.

## 분석 시점

- 버전 간 변경이 발생했을 때
- Draft 문서에 중대한 수정이 있을 때
- 새로운 요구사항 추가 시

## 분석 관점

Impact Analyzer는 다음을 분석합니다:

1. **영향 받는 문서** — 변경으로 인해 수정이 필요한 문서 목록
2. **영향 받는 Task** — 재작업이 필요한 Task 식별
3. **회귀 위험** — 기존 기능에 대한 영향도
4. **후속 작업** — 트레이서빌리티 업데이트, 릴리스 준비 상태

## 출력 형식

```
impact-analysis/
├── v1.0-to-v1.1-impact.md
├── prd-change-20260307-impact.md
└── ...
```

## 분석 결과 구조

```markdown
# 변경 영향 분석

- 분석 일시: YYYY-MM-DD
- 변경 범위: [변경된 문서/섹션]

## 영향 받는 문서

| 문서 | 영향도 | 필요 조치 |
|------|--------|-----------|

| api-spec.md | MEDIUM | 엔드포인트 추가 |

|---------|------|-----------|

2. ...
```
