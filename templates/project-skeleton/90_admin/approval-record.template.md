# Approval Record Template

승인 기록 작성 시 이 템플릿을 사용합니다.

---

## 파일명 규칙

`approval-{버전}-{YYYYMMDD}.md`

예시: `approval-v1.0-20260307.md`

---

## 템플릿

```markdown
# Approval Record

## 버전 정보

| 항목 | 값 |
|------|-----|
| 버전 | v{X.Y} |
| 승인 일시 | {YYYY-MM-DD HH:MM} |
| 승인자 | {이름} |
| 역할 | {Project Owner / Tech Lead / ...} |

---

## 승인 범위

### 문서 승인

| 문서 | 상태 | 리뷰 결과 | 비고 |
|------|------|-----------|------|
| prd.md | ✅ 승인 / ⚠️ 조건부 / ❌ 거부 | PASS | |
| architecture.md | | | |
| design.md | | | |
| api-spec.md | | | |
| db-schema.md | | | |

### Task 승인

| 항목 | 값 |
|------|-----|
| 총 Task 수 | {N}개 |
| READY 상태 | {N}개 |
| NEEDS_REVISION | {N}개 |
| 승인 여부 | ✅ 전체 승인 / ⚠️ 부분 승인 |

---

## 조건부 승인 사항

(조건부 승인 시 작성)

| 문서/Task | 조건 | 해결 기한 |
|-----------|------|-----------|
| {대상} | {조건 설명} | {vX.Y 또는 날짜} |

## 거부 사항

|-----------|-----------|-----------|
```
