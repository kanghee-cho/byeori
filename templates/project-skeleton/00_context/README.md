# Context (00_context)

이 폴더는 프로젝트의 **입력 자료**를 저장합니다.

## 목적

- 프로젝트 아이디어, 문제 정의, 초기 요구사항
- 참고 자료 (기존 문서, 이미지, 외부 링크)
- 배경 정보 및 제약 조건

## 사용 방법

1. 아이디어나 문제 설명을 자유 형식으로 작성
2. 기술 제약조건이 있다면 명시
3. 참고할 자료가 있다면 이 폴더에 배치

## 예시 파일

```
00_context/
├── idea.md              # 핵심 아이디어
├── constraints.md       # 기술 제약조건
├── references/          # 참고 자료
│   ├── competitor-analysis.md
│   └── market-research.pdf
└── stakeholder-notes.md # 이해관계자 메모
```

## idea.md 예시

```markdown
# 프로젝트 아이디어

## 문제
사용자들이 중고 물품을 안전하게 거래할 수 있는 플랫폼이 필요함.

## 목표
- 신뢰할 수 있는 거래 환경 제공
- 간편한 물품 등록 및 검색
- 실시간 채팅 기능

## 제외 범위
- 결제 시스템 (1차 버전)
- 배송 추적

## 기술 제약
- Backend: Node.js
- Cloud: AWS
```

## 관련 Agent
